# Shipment

###Model Yapısı

içerir. Bu değişkenler;

```php
const DELIVERY_TYPE_FREE = 'free';
```
```php
const DELIVERY_TYPE_BUYER_PAYS = 'buyerPays';
```
```php
const DELIVERY_TYPE_MERCHANT_PAYS = 'merchantPays';
```
